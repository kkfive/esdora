# @esdora/kit
> Zero-dependency TypeScript utilities: type guards, tree operations, URL parsing, number clamping, and promise helpers

## 安装
```sh
pnpm add @esdora/kit
```
零运行时依赖。

## 定位与依赖来源
Zero-dependency TypeScript utilities: type guards, tree operations, URL parsing, number clamping, and promise helpers
本包无外部运行时依赖。

## 子路径导出
- `./experimental`，`import { ... } from '@esdora/kit/experimental'`
