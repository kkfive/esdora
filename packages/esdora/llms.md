# esdora
> A comprehensive collection of development tools and utilities - Meta package that aggregates all Esdora packages

## 安装
```sh
pnpm add esdora
```

## 定位与依赖来源
A comprehensive collection of development tools and utilities - Meta package that aggregates all Esdora packages
本包无外部运行时依赖。
