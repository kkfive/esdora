# @esdora/biz
> Business utilities package with query string parsing (qs) and extensible for future business tools

## 安装
```sh
pnpm add @esdora/biz
```

## 定位与依赖来源
Business utilities package with query string parsing (qs) and extensible for future business tools
本包 re-exports 以下外部库：class-variance-authority, clsx, tailwind-merge。

## 子路径导出
- `./atom-css`，`import { ... } from '@esdora/biz/atom-css'`
- `./qs`，`import { ... } from '@esdora/biz/qs'`
