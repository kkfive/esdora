# @esdora/date
> A modern, immutable date utility library, powered by date-fns for reliability and enhanced with unique convenience functions.

## 安装
```sh
pnpm add @esdora/date
```

## 定位与依赖来源
A modern, immutable date utility library, powered by date-fns for reliability and enhanced with unique convenience functions.
本包 re-exports 以下外部库：date-fns。

## 子路径导出
- `./fp`，`import { ... } from '@esdora/date/fp'`
- `./locale`，`import { ... } from '@esdora/date/locale'`
